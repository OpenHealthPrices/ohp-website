The Data

All data is released under Attribution-NonCommercial-ShareAlike 4.0 International and can be used freely for non commercial use. If you have a commercial use case or want more frequent updates contact us.

Data is available in several formats. Each download contains a detailed readme describing the data format. Many of these downloads are very large. If you want help interpreting or integrating the data into your project contact us.
